# Snake and Ladder – C# Console App

This project implements the Snake and Ladder game in C# using multiple use cases (UC1–UC7).

## How to run

1. Open the solution in Visual Studio.
2. Make sure the `Snake-and-ladder` project is set as Startup Project.
3. Build and run (Ctrl + F5).

## Use Cases

- **UC1**: Initialize player at position 0.
- **UC2**: Show starting position and basic message.
- **UC3**: Roll the dice (1–6) and update position.
- **UC4**: Add options – No Play, Ladder, Snake.
- **UC5**: Enforce exact winning position 100 (cannot overshoot).
- **UC6**: Report position and dice roll count for each turn; show total rolls.
- **UC7**: Two-player game. On Ladder, same player gets another turn. Report which player wins and roll counts.
